Course Publishing NEW

POST /api/v1/courses/{id}/publish/

Publish an approved course. This endpoint changes the course status to PUBLISHED, making it visible to students and parents.

Precondition: The course must have APPROVED status.

Permissions: Only administrators or department heads can publish courses.

Notifications: Automatically sends notifications to trainers, learners, parents, and stakeholders.

Authorization: Bearer <your_access_token>
Response
Notifications
{ "status": "published", "new_status": "PUBLISHED", "course_id": 1 }

POST /api/v1/courses/{id}/submit/
Submit a draft course for review. Changes status from DRAFT to SUBMITTED.
Authorization: Bearer <your_access_token>
Response
{ "status": "submitted", "new_status": "SUBMITTED", "course_id": 1 }
